# Loan Default Prediction App

Explainable AI system for loan default prediction using XGBoost and SHAP.

## Live Demo
https://loan-default-abeer.streamlit.app

## Features
- Predicts loan default probability
- SHAP waterfall plot for explanation
- Fairness audit across demographic groups

## Tech Stack
Python, XGBoost, SHAP, Scikit-learn, Pandas, Streamlit

## Dataset
255,347 real loan records
